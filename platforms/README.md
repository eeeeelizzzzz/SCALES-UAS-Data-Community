# Site-Date UAS Activity Matrix (IOP)

This directory contains per-platform information for platforms active during SCALES. The table below summarizes which **platform IDs** appear at each named site and **IOP calendar day** (`YYYYMMDD`, UTC, from catalog filenames). **Empty cells (`-`)** mean no activity for that site/day in the source snapshot—not “no flights anywhere.” Site names and deployment context (including how downstream distance rings relate to operations) are easier to follow on the **[SCALES / Flight Week Coda](https://coda.io/d/SCALES-Flight-Week_dPqDysn4kSm/SCALES-Intro_suGBnNdS#_luWfxq7e)** campaign hub.

**Note:** Intercomparison-focused flights (e.g. **20240908**) are a subset of the campaign; this matrix reflects the broader IOP upload snapshot, not only intercomparison day.

**Source:** snapshot of UAS data uploaded to the WMO-UAS-DC database as of **20251208** (`SCALES_UAS_BY_IOP_final`–style catalogs).

*Footnote:* Some data archives include records under additional identifiers such as `ddfdbf` and `PIG00`. Verified records indicate `ddfdbf` was not part of the 20240908 intercomparison and appears only in two files on 20240909. The `ddfdbf` aircraft type is MM-670-21, and flight timing (and datata) does not match the OSU `MM100` flights, and OSU only operates one Meteodrone. The `ddfdbf` identifier may be an additional, unconfirmed label sometimes assigned to OSU's Meteomatics platform, or it could be something else entirely. Another platform `PIG00` does appear on intercomparison day, but no other time. The `PIG00` platform has not been clearly identified. It could be identifying a specific payload, but this is unclear.
